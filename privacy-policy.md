# 隱私權政策 Privacy Policy

**事實查核助手 Fact Check Assistant**
最後更新：2026年3月31日

## 我們收集哪些資料

本擴充功能**不會收集任何個人資料**。

- 不追蹤您的瀏覽紀錄
- 不儲存您查詢的文字到任何伺服器
- 不需要帳號登入

## 資料使用方式

當您選取文字並查核時：
1. 選取的文字會傳送至 Google Fact Check Tools API 進行查詢
2. 查詢結果只會暫時儲存在您的瀏覽器本機
3. 關閉瀏覽器後資料自動清除

## 第三方服務

本擴充功能使用 Google Fact Check Tools API，
請參閱 Google 隱私權政策：https://policies.google.com/privacy
